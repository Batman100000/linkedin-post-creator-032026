🔗 חבר לינקדאין – 3 אפשרויות

Zapier מחבר בין האפליקציה לינקדאין שלך בלי צורך בקוד. ✅ קל, מהיר, בחינם עד 100 פעולות/חודש.
1
כנס ל-zapier.com וצור חשבון חינמי
2
צור Zap חדש: Trigger = Webhook → Action = LinkedIn: Create Post
3
העתק את ה-Webhook URL והדבק כאן:
Buffer הוא כלי ניהול רשתות חברתיות. תומך בלינקדאין ומאפשר תזמון פוסטים. ✅ יש תוכנית חינמית.
1
כנס ל-buffer.com וחבר את חשבון הלינקדאין שלך
2
בהגדרות Buffer → API Access → צור Access Token
3
הדבק את ה-Token + מזהה הפרופיל כאן:
ללא חיבור API – האפליקציה תעתיק את הפוסט ותפתח את לינקדאין ישירות. הפרסום ידני.
1
לחץ "פרסם" → הפוסט יועתק אוטומטית ללוח
2
לינקדאין ייפתח בטאב חדש → הדבק ופרסם
3
תזמון: תקבל תזכורת push בדפדפן בשעה שתבחר
